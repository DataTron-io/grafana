# TagList

List of tags with predefined margins and positioning.

### Usage

```jsx
import { TagList } from '@grafana/ui';
const tags = ['datasource-test', 'gdev', 'mysql', 'mssql'];

<TagList tags={tags} onClick={tag => console.log(tag)} />
```
